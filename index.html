Projects

NC-News

This project was created as part of the Northcoders software development bootcamp, it consists of a back-end API for a news system similar to Reddit, where users can post articles on different topics. The users can also post comments on the articles. Articles and topics can receive positive and negative votes from users, depending on whether they liked the content or not.

  • React.Js frontend
  • CSS styling with Styled-components and Material UI
  • Node.Js with Express.Js backend
  • Test-Driven development withJEST
  • Relational database with PostgreSQL
  • MVC pattern design.

Labre-PR Backend

This is a Python API backend application for the Brazilian Amateur Radio Association, Labre-PR, website. The API provides endpoints for the website news system, and for the administration application, which handles associates registration and maintenance, annuity control and bureau control. The API makes uses of an admin token to access sensitive information only for the application adminstrators.

  • Python backend API
  • Using Blueprints for code organization.
  • SQLAlchemy for handling MySQL database queries.
  • Test-Driven development with PyTest.
Back-end repo Live hosted version

Books4U

A mobile App that uses AI to recommend books for you based on a user text input. You can make enquiries like "I just read the Maze runner series, can you recommend me something similar?" The App then uses MistralAI to find books similar to it, and then uses Google Books API to fetch the details for the books.

  • React Native with Expo frontend
  • CSS styling
  • Supabase with PostgreSQL backend
  • Artificial Inteligence responses with Mistral.Ai
